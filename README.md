# Homebrew Tap

Personal Homebrew tap for my applications.

## Installation

```bash
brew tap JohnVonDrashek/tap
```

## Available Casks

### Praymodoro

A prayerful Pomodoro timer with Catholic saints as desktop companions.

```bash
brew install --cask praymodoro
```

## Updating

```bash
brew update
brew upgrade --cask praymodoro
```
